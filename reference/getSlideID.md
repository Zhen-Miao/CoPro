# Get slide IDs from CoPro object

Get slide IDs from CoPro object

## Usage

``` r
getSlideID(object)

# S4 method for class 'CoProSingle'
getSlideID(object)

# S4 method for class 'CoProMulti'
getSlideID(object)
```

## Arguments

- object:

  A CoPro object (CoProSingle or CoProMulti)

## Value

For CoProMulti: the slideID vector; For CoProSingle: character(0)
