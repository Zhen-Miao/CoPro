# Get slide list from CoPro object

Get slide list from CoPro object

## Usage

``` r
getSlideList(object)

# S4 method for class 'CoProSingle'
getSlideList(object)

# S4 method for class 'CoProMulti'
getSlideList(object)
```

## Arguments

- object:

  A CoPro object (CoProSingle or CoProMulti)

## Value

For CoProMulti: the slideList vector; For CoProSingle: character(0)
