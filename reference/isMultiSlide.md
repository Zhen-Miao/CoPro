# Check if object is multi-slide

Check if object is multi-slide

## Usage

``` r
isMultiSlide(object)

# S4 method for class 'CoProSingle'
isMultiSlide(object)

# S4 method for class 'CoProMulti'
isMultiSlide(object)
```

## Arguments

- object:

  A CoPro object (CoProSingle or CoProMulti)

## Value

Logical indicating if object contains multiple slides
