# Get Self-Distance Matrix

Convenience function to retrieve self-distance matrices computed by
computeSelfDistance.

## Usage

``` r
getSelfDistMat(object, cellType, slide = NULL, verbose = TRUE)
```

## Arguments

- object:

  A CoPro object

- cellType:

  Cell type name

- slide:

  Slide ID (for CoProMulti objects)

- verbose:

  Whether to print error messages

## Value

Self-distance matrix for the specified cell type
